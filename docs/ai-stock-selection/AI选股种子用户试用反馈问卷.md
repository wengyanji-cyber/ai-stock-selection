# AI选股种子用户试用反馈问卷

## 1. 文档目的

本问卷用于种子用户试用期结束后的反馈收集，帮助团队评估产品是否具备真实留存价值、理解价值和付费潜力。

## 2. 使用建议

建议在用户连续试用 7 到 14 天后发放本问卷，并结合行为数据一起看，避免只听用户主观评价。

## 3. 基本信息

1. 你平时主要交易哪类股票。
2. 你大概多久看一次盘。
3. 你在试用期间大概使用了本产品几天。
4. 你最常用的页面是哪一个。

## 4. 使用频率反馈

1. 你是否几乎每天都会打开产品。
2. 你通常在什么时间使用产品。
3. 你主要是在盘前、盘中还是盘后使用。
4. 你最常点开的内容是什么。

## 5. 功能价值反馈

请对以下功能按 1 到 5 分打分：

1. 盘前一句话。
2. 热点板块。
3. 今日候选池。
4. 个股诊断。
5. 自选观察。
6. 风险提示。
7. 每日复盘。

## 6. 内容理解反馈

1. 你是否能快速看懂每只股票的一句话结论。
2. 你是否理解观察区间、支撑、压力和止损这些点位。
3. 你觉得风险提示是否足够清楚。
4. 你是否觉得内容过多、过少，还是刚好。

## 7. 决策帮助反馈

1. 产品是否帮你缩短了选股和看盘时间。
2. 产品是否帮你减少了信息焦虑。
3. 产品是否帮你更快判断一只股票能不能继续看。
4. 产品是否让你更愿意按计划而不是情绪交易。

## 8. 候选池反馈

1. 你觉得每天候选股数量是否合适。
2. 你更喜欢重点候选还是扩展观察。
3. 你是否认为候选股理由可信。
4. 你是否更想看更少候选还是更多候选。

## 9. 诊股反馈

1. 你是否主动输入过自己的股票做诊断。
2. 你觉得诊断结果最有帮助的部分是什么。
3. 你觉得诊断结果最不足的部分是什么。
4. 你希望诊股结果增加什么内容。

## 10. 风险与信任反馈

1. 你是否觉得产品表达方式可信。
2. 你是否担心它只是另一种荐股工具。
3. 哪种表达会让你更信任这个产品。
4. 哪种表达会让你立刻反感或不信任。

## 11. 付费意愿反馈

1. 试用结束后，你是否愿意继续使用。
2. 如果愿意，你更能接受月付、季付还是先继续试用。
3. 你愿意为这个产品支付的价格区间大致是多少。
4. 影响你是否付费的最关键因素是什么。

## 12. 开放式问题

1. 这个产品最有价值的一点是什么。
2. 这个产品最需要改进的一点是什么。
3. 如果只能保留一个功能，你希望保留哪一个。
4. 如果还能新增一个功能，你最希望加什么。

## 13. 结果分析建议

建议把问卷结果与用户行为数据合并分析，重点关注：

1. 高使用频次用户最常用什么功能。
2. 愿意付费用户最看重什么价值。
3. 不愿付费用户主要是因为效果、价格、信任还是理解成本。
4. 哪些模块打分高但实际使用低。

## 14. 结论

问卷的目标不是收集好评，而是识别哪些功能值得继续投，哪些表达方式需要马上调整，哪些承诺用户其实并不买账。
