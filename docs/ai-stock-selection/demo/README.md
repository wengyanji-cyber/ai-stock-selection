# AI选股演示版说明

## 1. 位置

演示文件位于：

1. [index.html](index.html)
2. [data.js](data.js)
3. [app.js](app.js)
4. [trial.html](trial.html)

6. [login.html](login.html)
7. [account.html](account.html)
8. [ops.html](ops.html)
9. [shared.css](shared.css)

## 2. 用途

这是一个静态 HTML 演示页，用于快速展示 AI 选股产品的核心信息结构和页面组织方式。

覆盖五个核心视图：

1. 首页。
2. 候选池。
3. 个股诊断。
4. 自选观察。
5. 复盘。

并新增两张商业化演示页面：

1. 试用开通页。
2. 订阅方案页。

并补充两张用户路径页面：

1. 登录注册页。
2. 用户中心页。

并补充一张内部运营页面：

1. 运营后台概览页。

## 3. 特点

1. 不依赖前端工程。
2. 不依赖构建工具。
3. 可直接本地打开查看。
4. 内置模拟数据，适合演示和评审。
5. 支持候选池筛选、个股诊断切换、自选状态过滤。
6. 试用、订阅、登录、留存、运营路径可串联演示。
7. 业务页样式已抽离到 shared.css，便于继续扩展。

## 4. 说明

当前演示版重点是页面信息结构，不代表真实行情系统，也不接真实数据接口。

## 5. 当前交互

当前演示页已支持以下交互：

1. 顶部页签切换。
2. 候选池按板块、优先级、风险等级筛选。
3. 候选卡片跳转到对应诊股视图。
4. 个股诊断按代码或名称切换。
5. 自选观察按状态筛选。
6. 补充试用开通和订阅方案演示路径。
7. 补充登录注册与用户中心演示路径。
8. 补充运营后台概览路径。
9. 主演示页可直接跳转完整业务链路。
