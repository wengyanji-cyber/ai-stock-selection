# AI选股项目重构版商业方案

## 1. 项目定位

项目名称可暂定为：AI短线投研助手。

一句话定位：

面向普通投资者的 AI 短周期投研辅助产品，聚焦热点板块跟踪、候选股筛选、个股风险识别、交易计划生成与持仓观察，不代客操作，不承诺收益。

这句话同时解决四个问题：

1. 讲清楚服务对象是谁。
2. 讲清楚是辅助工具，不是代客操盘。
3. 讲清楚是短周期，不是全资产配置。
4. 讲清楚产品价值，而不是空喊 AI。

## 2. 市场机会

散户在股票市场长期存在四个核心痛点：

1. 信息过载。每天公告、新闻、研报、社媒信息太多，无法高效筛选。
2. 决策低效。知道热点很多，但不知道该看哪只、什么位置看、什么情况下不该碰。
3. 情绪化交易。追涨杀跌、从众跟风、拿不住、止损不执行，是散户长期亏损的重要原因。
4. 缺少持续陪伴。大多数产品只提供数据，不提供结构化结论、交易计划和后续追踪。

现有产品两端都不满足这个需求：

1. 免费工具型产品强在信息检索，弱在结构化结论和连续跟踪。
2. 机构量化型产品强在数据和模型，弱在散户可获得性与可支付性。

因此，中间存在一块空白市场：不是做暴富神器，而是做散户用得懂、看得懂、每天愿意打开的 AI 投研辅助产品。

## 3. 目标用户

首阶段聚焦一类用户，不做全打。

建议优先主打：半懂技术面的活跃散户。

用户画像：

1. 有自选股和看盘习惯。
2. 懂一点 K 线、均线、量能、支撑压力。
3. 喜欢热点、题材、龙头。
4. 愿意为效率和结论付费。
5. 不完全相信老师带单，但愿意接受 AI 辅助判断。

暂不建议优先触达的两类用户：

1. 完全小白，教育成本太高。
2. 职业短线交易者，要求太高，极易失望。

## 4. 核心价值主张

产品不卖神奇预测，而卖四个价值：

1. 少看 90% 噪音，只看最值得关注的少量标的。
2. 每只股票给出简洁清晰的关注理由和风险点。
3. 给出可执行的观察区间、压力位、止损位和失效条件。
4. 持续追踪而不是一次性推荐。

对用户最有效的一句价值表达：

每天只看少量高质量机会，让普通投资者更快看懂市场、更少凭情绪交易。

## 5. 产品边界

必须明确写清楚做什么和不做什么。

做什么：

1. 板块热点跟踪。
2. 候选股筛选与排序。
3. 个股趋势与风险诊断。
4. 交易计划建议。
5. 持仓观察与风险提醒。
6. AI 问答解释。

不做什么：

1. 不承诺收益。
2. 不代客操作账户。
3. 不进行自动下单。
4. 不做保本保收益宣传。
5. 不以高胜率暴利为核心营销点。
6. 不直接开展私募募集或面向散户资金管理业务。

## 6. 产品核心功能

首版建议只做五个功能模块。

### 6.1 每日盘前一句话

输出当天市场方向、重点板块、主要风险。

### 6.2 热点板块雷达

展示 3 到 5 个重点方向，并附热度变化、龙头候选、风险提醒。

### 6.3 候选股池

每天输出 5 到 10 只股票，给出：

1. 买入观察区间。
2. 支撑。
3. 压力。
4. 止损。
5. 1 到 3 条理由。
6. 失效条件。
7. 风险等级。

### 6.4 个股极简诊断

用户输入股票代码，返回：

1. 趋势状态。
2. 主力强弱。
3. 量价关系。
4. 风险事件。
5. 建议观察方式。

### 6.5 持仓观察

围绕用户自选和持仓，给出：

1. 转强。
2. 转弱。
3. 跌破支撑。
4. 板块退潮。
5. 热点切换等信号。

## 7. 与现有股票软件的差异化

差异化不应写成我们也有 AI，而应写成以下五条：

1. 输出少量高质量结论，而不是海量数据。
2. 更强调短周期交易计划而不是静态研报。
3. 更强调看得懂的解释和后续跟踪。
4. 更贴近散户语言，不堆术语。
5. 在热点与个股之间建立联动关系。

## 8. 商业模式

前期建议只做订阅制，不碰高风险分成模式。

收费结构建议：

1. 免费版：盘前一句话、部分板块摘要、部分个股诊断。
2. 标准会员：完整候选池、个股诊断、持仓观察、风险提示、AI 问答。
3. 高级会员：增加深度诊断、重点股跟踪、板块轮动复盘、盘中异动提醒。

试用方式：

1. 3 天公开体验。
2. 14 天深度试用。
3. 再转月付或季付。

不建议前期主推年费，因为产品尚未跑出稳定留存。

## 9. 变现路径重构

原方案：试用收费 -> 帮用户炒股收佣 -> 成立私募。

重构后：订阅产品 -> 与持牌机构合作输出能力 -> 如未来进入资管业务则单独立项。

更具体地说：

1. 第一阶段，靠订阅收入验证产品成立。
2. 第二阶段，如用户量、留存、能力成熟，可与持牌券商、投顾、财富机构合作，输出选股、诊股、陪伴能力。
3. 第三阶段，如果未来考虑资管或私募，必须单独组建合规、风控、法务、投研体系，不与散户工具混用。

## 10. 合规原则

建议单独做一页内部口径，统一所有对外表达。

高风险表达，不建议对外使用：

1. 稳赚。
2. 高胜率。
3. 暴富。
4. 快钱。
5. 跟着买就行。
6. 帮你炒股。
7. 代客操盘。
8. 保本。
9. 收益承诺。
10. 媲美机构稳赚系统。

更稳妥的表达：

1. 决策辅助。
2. 候选筛选。
3. 风险识别。
4. 交易计划建议。
5. 热点跟踪。
6. 短周期观察工具。

## 11. 项目成败关键

这个项目不是比谁讲故事大，而是比谁先跑通以下五件事：

1. 是否能持续输出少量高质量候选。
2. 用户是否看得懂并愿意执行。
3. 产品是否能形成日更使用习惯。
4. 试用后是否有真实续费。
5. 风险控制和文案边界是否清晰。

如果这五条里前三条不成立，商业模式再漂亮也没用。

## 12. 对外版项目介绍范文

我们正在打造一款面向普通投资者的 AI 短周期投研辅助产品。产品聚焦热点板块跟踪、候选股筛选、个股风险识别和交易计划生成，通过结构化结论、关键点位和风险提示，帮助用户更高效地理解市场、减少情绪化交易。首阶段专注 A 股热点行业，先在少数板块中验证模型与产品价值，再逐步扩展能力边界。
