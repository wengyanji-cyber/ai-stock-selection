# 个股诊断设计

## 1. 功能定位

### 1.1 目标
为用户提供专业、全面、可视化的个股诊断服务，帮助用户快速了解股票的投资价值和风险。

### 1.2 核心价值
- **专业性**：多维度技术分析 + 基本面评估
- **可视化**：图表展示 + 评分系统
- **可操作性**：明确的支撑/压力/止损位 + 操作建议
- **个性化**：根据用户偏好调整诊断权重

## 2. 数据模型

### 2.1 诊断结果模型 (StockDiagnosis)

```typescript
interface StockDiagnosis {
  // 基础信息
  stockCode: string
  stockName: string
  sector: string
  tradeDate: Date
  
  // 综合评分
  overallScore: number           // 0-100分
  
  // 多维度评分
  dimensionScores: {
    technical: number            // 技术面 0-100
    fundamental: number          // 基本面 0-100
    capital: number              // 资金面 0-100
    sentiment: number            // 情绪面 0-100
  }
  
  // 趋势判断
  trend: {
    direction: 'up' | 'down' | 'neutral'
    strength: 'strong' | 'medium' | 'weak'
    period: 'short' | 'medium' | 'long'
  }
  
  // 技术位
  technicalLevels: {
    support: number[]            // 支撑位（多个）
    pressure: number[]           // 压力位（多个）
    stopLoss: number             // 止损位
    target: number               // 目标价
  }
  
  // 技术指标
  indicators: {
    ma: { ma5: number; ma10: number; ma20: number; ma60: number }
    macd: { dif: number; dea: number; histogram: number }
    kdj: { k: number; d: number; j: number }
    rsi: number
    boll: { upper: number; middle: number; lower: number }
  }
  
  // 资金流向
  capitalFlow: {
    mainInflow: number           // 主力净流入（万元）
    retailInflow: number         // 散户净流入
    northInflow: number          // 北向资金
    largeOrder: number           // 大单净流入
  }
  
  // 基本面
  fundamental: {
    pe: number                   // 市盈率
    pb: number                   // 市净率
    roe: number                  // 净资产收益率
    revenueGrowth: number        // 营收增长率
    profitGrowth: number         // 利润增长率
  }
  
  // 诊断结论
  conclusion: {
    summary: string              // 一句话总结
    reasons: string[]            // 判断理由
    risks: string[]              // 风险提示
    action: string               // 操作建议
  }
  
  // 相似股票
  similarStocks: SimilarStock[]
  
  // 历史诊断
  history: DiagnosisHistory[]
}

interface SimilarStock {
  code: string
  name: string
  similarity: number             // 相似度 0-100
  correlation: number            // 相关系数
}

interface DiagnosisHistory {
  date: Date
  score: number
  trend: string
  action: string
}
```

## 3. 诊断维度

### 3.1 技术面分析 (权重: 40%)

#### 趋势分析
- **均线系统**：MA5/MA10/MA20/MA60排列
- **趋势方向**：上涨/下跌/震荡
- **趋势强度**：强/中/弱

#### 技术指标
| 指标 | 买入信号 | 卖出信号 | 权重 |
|------|---------|---------|------|
| MACD | DIF上穿DEA，红柱放大 | DIF下穿DEA，绿柱放大 | 25% |
| KDJ | K上穿D，J<20 | K下穿D，J>80 | 20% |
| RSI | RSI<30超卖 | RSI>70超买 | 15% |
| 布林带 | 触及下轨反弹 | 触及上轨回落 | 20% |
| 成交量 | 放量上涨 | 缩量下跌 | 20% |

#### 技术位分析
- 支撑位：近期低点、均线、黄金分割
- 压力位：近期高点、均线、整数关口
- 止损位：支撑位下方3-5%

### 3.2 基本面分析 (权重: 25%)

#### 估值水平
- PE/PB 历史分位
- 行业对比
- 成长性评估

#### 财务健康
- ROE > 15%：优秀
- 营收增长 > 20%：高增长
- 利润增长 > 20%：高盈利

### 3.3 资金面分析 (权重: 25%)

#### 主力资金
- 主力净流入 > 1000万：强势
- 连续3日净流入：持续流入
- 大单占比 > 30%：机构关注

#### 北向资金
- 北向净流入：外资看好
- 北向持股比例：外资认可度

### 3.4 情绪面分析 (权重: 10%)

#### 市场情绪
- 板块热度排名
- 新闻舆情分析
- 搜索热度

## 4. 评分算法

### 4.1 综合评分计算

```typescript
function calculateOverallScore(diagnosis: StockDiagnosis): number {
  const weights = {
    technical: 0.40,
    fundamental: 0.25,
    capital: 0.25,
    sentiment: 0.10,
  }
  
  // 各维度得分
  const technicalScore = calculateTechnicalScore(diagnosis)
  const fundamentalScore = calculateFundamentalScore(diagnosis)
  const capitalScore = calculateCapitalScore(diagnosis)
  const sentimentScore = calculateSentimentScore(diagnosis)
  
  // 加权平均
  const overall = 
    technicalScore * weights.technical +
    fundamentalScore * weights.fundamental +
    capitalScore * weights.capital +
    sentimentScore * weights.sentiment
  
  return Math.round(overall)
}

// 技术面评分
function calculateTechnicalScore(d: StockDiagnosis): number {
  let score = 50  // 基准分
  
  // 趋势加分
  if (d.trend.direction === 'up') score += 20
  if (d.trend.strength === 'strong') score += 10
  
  // MACD
  if (d.indicators.macd.histogram > 0) score += 10
  
  // KDJ
  if (d.indicators.kdj.j < 20) score += 10  // 超卖反弹
  if (d.indicators.kdj.j > 80) score -= 10  // 超买回调
  
  // RSI
  if (d.indicators.rsi > 50 && d.indicators.rsi < 70) score += 5
  
  // 布林带
  const price = d.technicalLevels.support[0]  // 当前价
  if (price < d.indicators.boll.lower) score += 10  // 下轨支撑
  if (price > d.indicators.boll.upper) score -= 10  // 上轨压力
  
  return Math.max(0, Math.min(100, score))
}

// 资金面评分
function calculateCapitalScore(d: StockDiagnosis): number {
  let score = 50
  
  // 主力净流入
  if (d.capitalFlow.mainInflow > 5000) score += 30
  else if (d.capitalFlow.mainInflow > 1000) score += 20
  else if (d.capitalFlow.mainInflow > 0) score += 10
  else score -= 10
  
  // 北向资金
  if (d.capitalFlow.northInflow > 1000) score += 10
  
  return Math.max(0, Math.min(100, score))
}
```

### 4.2 评级标准

| 分数 | 评级 | 建议 |
|------|------|------|
| 90-100 | S | 强烈买入 |
| 80-89 | A | 买入 |
| 70-79 | B | 持有/关注 |
| 60-69 | C | 观望 |
| <60 | D | 回避 |
